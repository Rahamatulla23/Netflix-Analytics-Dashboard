# 📊 Netflix Analytics Dashboard

A Power BI dashboard analyzing Netflix content, users, and watch history.  
It provides insights into **user engagement, content growth, subscription distribution, and viewing behavior**.

## 🔎 Key Features
- KPIs: Total Content, Total Users, Total Watch Time  
- Content Growth by Year (line chart)  
- Genre Popularity (bar chart of watch time by genre)  
- Device Usage (bar chart of most used devices)  
- Subscription Distribution (donut chart: Basic vs Standard vs Premium)  
- Country Engagement (map visualization of watch time by country)  
- User Age vs Engagement (scatter plot)  
- Usage Trend Over Time (line chart)

## 🛠 Tech Stack
- Power BI Desktop  
- DAX Measures for KPIs and aggregations  
- Relational modeling between content, users, and watch history  

## 🚀 How to Use
1. Clone the repository.  
2. Open the `.pbix` file in **Power BI Desktop**.  
3. Explore interactive visuals using slicers (genre, subscription type, device, year).  

## 📈 Insights
This dashboard helps answer:
- Which genres are most watched  
- How Basic vs Premium users differ  
- Which devices dominate viewing  
- Which countries engage the most  
- How Netflix content has grown over time
